# Variante B — Editorial Signature (Multipágina)

Propuesta premium multipágina para elevar percepción de marca y conversión consultiva.

## Páginas

- `index.html` (narrativa principal + scrolltelling SVG)
- `coleccion.html`
- `servicios.html`
- `contacto.html`

## Incluye

- Loader global en cada página
- GSAP + ScrollTrigger + Lenis
- Scrolltelling con SVG animado en inicio
- UX editorial y navegación multipágina

## Abrir

Servir la carpeta en un servidor estático y abrir `index.html`.
